# Interview Talking Points

- I built this as a public-safe portfolio project using synthetic data.
- The goal was to show a practical junior-level workflow, not to claim production ownership.
- The most important decision was making the project understandable in 30 seconds.
- I separated public sample data from any real employer, client, academic, personal, or job-search data.
- Technical stack: time series, pandas, scikit-learn, forecasting metrics.
- Recruiter signal: Juan links workforce forecasting concepts to a transparent Python model recruiters can inspect quickly.
- If I had more time, I would add tests, screenshots, deployment notes, and a small CI workflow.
